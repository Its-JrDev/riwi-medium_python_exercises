# Python Practice Exercises

## Description

This repository contains a collection of Python exercises focused on practicing fundamental programming concepts.
The problems simulate real-life situations such as stores, gyms, cafeterias, parking lots, and service businesses.

The objective of these exercises is to strengthen logical thinking and reinforce the use of basic programming structures in Python.

## Topics Practiced

The exercises cover several core programming concepts:

* **Variables** – storing values such as prices, ages, names, and quantities.
* **Conditional statements (`if`, `elif`, `else`)** – making decisions based on conditions.
* **Loops (`for`, `while`)** – repeating actions to process multiple inputs.
* **Counters** – counting occurrences of events (for example, how many times a product was requested).
* **Accumulators** – summing totals such as sales, payments, or minutes trained.
* **Functions** – organizing code into reusable blocks to improve readability and structure.
* **Lists** – storing collections of values like prices or product names.
* **Dictionaries** – mapping categories to values (for example, products and their prices).
* **Iterators** – traversing collections using loops.
* **Comparisons with strings** – evaluating user input such as product names or services.
* **Arithmetic operations** – performing calculations like totals, averages, and discounts.

## Exercise Categories

The exercises are divided into two difficulty levels:

### Basic Exercises

These focus on understanding conditionals, loops, and simple calculations.

Examples include:

* Ice cream shop flavor counter
* Gym access by age
* Cafeteria purchase calculator
* Cinema ticket pricing by age
* Pet store food recommendation
* Parking lot hourly charge
* Hair salon schedule classification
* Counting expensive sports products
* Spa service validation
* Dance academy attendance classification

### Intermediate Exercises

These involve more complex logic with multiple variables, counters, and accumulators.

Examples include:

* Ice cream shop sales system for multiple clients
* Gym weekly performance tracking
* Cafeteria discount system
* Cinema room capacity control
* Parking lot vehicle management
* Pet store sales by category
* Hair salon daily service report
* Language center student evaluation
* Sports clothing inventory monitoring

## Learning Goals

By completing these exercises, the following skills are developed:

* Logical problem solving
* Structuring programs with clear control flow
* Handling user input and data validation
* Tracking and summarizing information with counters and accumulators
* Writing cleaner and more organized Python code
